## 문제 정의
> 번들 포장 불량으로 인해 발생하는 공정 이상은 전체 설비 불가동 시간의 55%를 차지하는 심각한 문제였습니다.
>
> 알람 데이터 레이블링 문제와 제품 단위 별 상이한 데이터 분포로 인해 모델 개발에 추가적인 고려가 필요한 상황이었습니다.
>
> 이에 대응하여, 비지도 학습 기반의 LSTM-AE 이상탐지 모델을 개발하고, MLOps 관점에서 모델을 실시간으로 배포하였습니다.
